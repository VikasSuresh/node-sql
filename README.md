## Using SQL in Node

Trying to Use Sql in node via postgres-package or using Sequelize ORM

## Installation
```bash
npm i
```

## To run the server
```bash
npm start 
```
 
## Description:

1. It uses Postgres Sql to connect to Postgres-server in this Postgres-Docker is used.
2. Basic Crud ops using postgres.
3. Sequelize - An ORM.
